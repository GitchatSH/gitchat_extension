# Top GitHub Trending Repo & People

Discover trending GitHub repos and developers, connect with devs, and chat — all without leaving your editor.

Works with VS Code, Cursor, Windsurf, Void, OpenCode, and any IDE supporting VS Code extensions.

## Features

- **Trending Repos** — Browse what's hot on GitHub right now
- **Trending People** — Discover top developers
- **Activity Feed** — See what the community is building
- **Follow & Star** — Follow devs and star repos inline
- **Messaging** — Chat with developers directly in your editor
- **Notifications** — Real-time alerts for follows, stars, and messages
- **Profiles** — View developer profiles with star power scores
- **Search** — Find repos and people from the command palette

## Getting Started

1. Install the extension from the marketplace
2. Click the fire icon in the activity bar
3. Browse trending repos and people (no login required)
4. Sign in with GitHub to unlock social features (follow, message, feed)

## Commands

Open the command palette (`Cmd+Shift+P`) and type "Trending":

| Command | Description |
|---------|-------------|
| `Trending: Sign In` | Sign in with GitHub |
| `Trending: Search` | Search repos & people |
| `Trending: Open Inbox` | Open your messages |
| `Trending: Message @user` | Start a conversation |

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+G G` | Toggle sidebar |
| `Cmd+Shift+G T` | Browse trending |
| `Cmd+Shift+G M` | Open inbox |
